# :penguin: Ubuntu

- 操作系统：Ubuntu 20.04.6 LTS

## 配置步骤

参考 https://polyscope.run/building/

1. 将项目下载到本地：

    ```
    git clone --recurse-submodules https://github.com/nmwsharp/polyscope.git
    ```

2. 根据实际项目需要，添加该项目目录到对应的 CMakeLists..txt 文件中即可，具体参考 https://polyscope.run/building/
