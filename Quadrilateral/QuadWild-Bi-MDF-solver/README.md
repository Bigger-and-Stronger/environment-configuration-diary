# :penguin: Ubuntu

- 操作系统：Ubuntu 20.04.5 LTS
- SSH IDE：CLion 2024.3.4

## 配置步骤

1. 将项目下载到本地，在终端中输入：

    ```
    git clone https://github.com/cgg-bern/quadwild-bimdf.git
    ```

    并进入该项目目录：

    ```
    cd quadwild-bimdf
    ```
